# Task Management Context

This directory contains context files for task management system used by TaskManager subagent.

## Structure

- **navigation.md** - This file
- **guides/** - How-to guides for task management
- **lookup/** - Command reference and lookup tables
- **standards/** - JSON schema and standards

## Related Files

- TaskManager agent: `.opencode/agent/subagents/core/task-manager.md`

## Usage

Load these context files when working with TaskManager or using the task management CLI.
