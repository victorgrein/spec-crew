# Governance

## What This Pack Covers

Standards and policy guidance that keep CrewAI projects consistent and maintainable.

- Code quality and naming standards
- Project structure conventions
- CLI usage policy and command hygiene
- Task tracking and dependency governance

## Use This Pack When

- The request is documentation, standards, or policy oriented.
- Teams need consistency checks before merge/release.
- Task sequencing and dependency integrity must be tracked.

## Normalized Trigger Phrases

- project standards
- code quality rules
- project structure guide
- task dependency tracking
- cli workflow policy

## Operating Rules

- Keep standards actionable and audit-friendly.
- Keep policy language aligned with canonical commands and agents.
- Keep governance checks lightweight but deterministic.
